# MongoDB-Fetcher-Client-Side-
Fabric client-side Minecraft 1.21.11 mod that adds /info &lt;name> to look up a player’s region and tier from a MongoDB collection and display it in chat. Configurable database, collection, and field names make it easy to connect to your own player ranking data.
